# Simple-HTTP-Server
A simple HTTP server for Windows

## Tech
- C/C++
- Winsock2

## Features
- GET
  - file type support:
    - text/plain
    - text/html
    - image/jpeg
    - image/png
- POST
  - test (only one case)
    - action:dopost

## How to run
- git clone
- cd release
- run Server.exe
- Open browser, and input "localhost:3367".(or your local ip address with port 3367)
